# Documentation Assets

This directory contains assets for the documentation site.

## Logo and Favicon

To add a custom logo and favicon to the documentation:

1. Add your logo image to `docs/assets/images/logo.png`
2. Add your favicon to `docs/assets/images/favicon.png`
3. Uncomment the following lines in `mkdocs.yml`:
   ```yaml
   logo: assets/images/logo.png
   favicon: assets/images/favicon.png
   ```

## Recommended Sizes

- **Logo**: PNG format, transparent background, approximately 128x128 pixels or larger
- **Favicon**: PNG format, 32x32 or 64x64 pixels
